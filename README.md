# Lancement du projet
Pour lancer le projet dans son intégralité, seule la commande suivante est nécessaire : 
```bash
python main.py
```
Cela créera donc un fichier dans le dossier ```output/```. 

L'affichage du dashboard se fait à partir de la fonctionnalité "Go Live" intégrée à l'IDE Visual Studio Code ou encore en allant à la racine du dossier pour sélectionner le fichier directement.
